# app template

This is a simple template for a Node.js app with TypeScript ESM. 

## Install Dependencies

```bash
pnpm install
```

## Run Tests

```bash
pnpm test
```

or use wallaby.js to run tests in watch mode. See [wallaby.js](https://wallabyjs.com/) for more information.

## Run App

```bash
pnpm start
```

## Run Dev Server

```bash
pnpm dev
```
